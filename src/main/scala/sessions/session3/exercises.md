
📝 **Exercise 1**:

Write a function `processNumbers` that:
1. Takes a `List[Int]` as input
2. Keeps only the even numbers
3. Multiplies each element by 3
4. Returns the sum of the resulting elements

**Example**

```scala
   val nums = List(1, 2, 3, 4, 5, 6)
   val result = processNumbers(nums)
   println(result) // 36
```
