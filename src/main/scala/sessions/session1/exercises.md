
📝 **Exercise 1**: simple display

Write a program that:

- Declares a variable `name`
- Prints `Hello <name>` using string interpolation
- Prints the first 10 integers using a `for` loop

🎮 **Exercise 2**: mystery number game

Write a program that:

- Picks a random number between 0 and 100
- Asks the user to guess it
- After each attempt, indicates whether the number is too small or too big
- Prints the number of attempts needed to find the answer

Use `scala.io.StdIn.readLine` to read user input
